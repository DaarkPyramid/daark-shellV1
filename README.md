## Description

This script demonstrates the basics of creating a reverse shell for learning and testing in a **controlled and authorized environment** (like a lab or virtual machines).  
It is intended to help understand how shell connections work.

---

## Features

- Educational reverse shell implementation
- Works in a safe lab environment
- Placeholder IP and PORT values (change for lab testing only)

---

## Usage

1. Update the script with your **lab IP** and **PORT**.  
2. Run a listener on your testing machine.  
3. Run the script in a **safe and controlled environment**.

```bash
# Example:
IP = "LAB_IP_HERE"
PORT = 4444
